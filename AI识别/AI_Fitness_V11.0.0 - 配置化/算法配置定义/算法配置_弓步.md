# 算法配置：弓步

## 1. 基本信息
- **动作ID**：`lunge`
- **动作名称**：弓步
- **版本**：V1.0
- **描述**：检测弓步蹲过程中的膝关节稳定性、下蹲幅度及躯干中立位。

## 2. 全局逻辑控制
- **显示模式**：互斥
- **优先级排序**：升序
- **优先级压制**：开启
- **优先级挤占**：关闭
- **纠错交互触发频率**：2

## 3. 虚拟计算点池配置
- **髋部中点**：
   - *算法*：`中点`
   - *输入*：左髋, 右髋
- **肩部中点**：
   - *算法*：`中点`
   - *输入*：左肩, 右肩
- **膝盖中点**：
   - *算法*：`中点`
   - *输入*：左膝, 右膝
- **前脚底中心**：
   - *算法*：`中点` (需结合状态机判定前脚，此处定义通用逻辑)
   - *输入*：左踝, 左脚尖 (或 右踝, 右脚尖)
- **前膝理想投影**：
   - *算法*：`投影`
   - *输入*：源=前膝, 参考=前脚底中心
   - *参数*：方向=垂直

## 4. 指标池配置
- **髋膝垂直距离**：
  - *算法*：`距离`
  - *输入*：髋部中点, 膝盖中点
  - *参数*：模式=垂直
- **下蹲深度**：
  - *算法*：`压缩率`
  - *输入*：当前=髋膝垂直距离, 基准=站立基准
- **双膝间距**：
  - *算法*：`距离`
  - *输入*：左膝, 右膝
- **双髋间距**：
  - *算法*：`距离`
  - *输入*：左髋, 右髋
- **膝内扣比例**：
  - *算法*：`比例`
  - *输入*：分子=双膝间距, 分母=双髋间距
- **躯干长度**：
  - *算法*：`距离`
  - *输入*：肩部中点, 髋部中点
  - *参数*：模式=欧氏
- **肩髋水平偏移**：
  - *算法*：`距离`
  - *输入*：肩部中点, 髋部中点
  - *参数*：模式=水平
- **躯干侧倾偏移率**：
  - *算法*：`偏移率`
  - *输入*：距离=肩髋水平偏移, 参考=躯干长度

## 5. 动态基准池配置
- **站立基准**：
  - *名称*：站立基准高度
  - *来源指标*：髋膝垂直距离
  - *采集逻辑*：
    - 阻尼：0.01
    - 衰减：0.9995

## 6. 状态机配置
- **准备**：
  - 说明：双脚并拢站立，动作就绪
  - 示例：`进入: 复位状态停留时长 > 0.5s`
- **向心 (下蹲)**：
  - 说明：下蹲阶段
  - 示例：`下蹲深度 < 0.85`
- **离心 (起立)**：
  - 说明：起立阶段
  - 示例：`下蹲深度 > 0.90`
- **进入纠错区间**：
  - 说明：进入弓步底部
  - 示例：`下蹲深度 < 0.35`
- **离开纠错区间**：
  - 说明：离开弓步底部
  - 示例：`下蹲深度 > 0.38`
- **复位**：
  - 说明：动作结束
  - 示例：`进入: 下蹲深度 > 0.92`
- **超时熔断**：
  - 示例：`时长 > 6.0s AND 下蹲深度 > 0.70`

## 7. 错误项配置架构

### 错误项 A：膝内扣
- **ID**：`valgus`
- **优先级**：`P1`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`膝内扣比例`
  - 阈值：`< 1.0`
  - 容忍度：`严格 < 1.1 / 标准 < 1.0 / 宽松 < 0.9`
  - 引导词:
    - `< 1.0 : "膝盖向外打开，对准脚尖"`
    - `>= 1.0 : "膝盖方向正确，保持住"`
- **判定策略配置**：
  - 检测时机：纠错区间
  - 单双侧逻辑：整体
  - 时序防抖：5帧
- **判定模式**：`过程修正`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `前膝`，方向 `向外`
      - 样式：红色
      - 动效：`弹性`
    - 元素名称：
      - 类型：`参考线`
      - 定义：`前脚底中心` -> `前膝理想投影`
      - 样式：绿色、虚线

### 错误项 B：下蹲幅度不足
- **ID**：`depth`
- **优先级**：`P2`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`下蹲深度`
  - 阈值：`< 0.20`
  - 容忍度：`严格 < 0.15 / 标准 < 0.20 / 宽松 < 0.25`
  - 引导词:
    - `> 0.20 : "后膝下沉，再蹲深一点"`
    - `<= 0.20 : "深度正好，前脚掌蹬地起立"`
- **判定策略配置**：
  - 检测时机：纠错区间
  - 单双侧逻辑：整体
  - 时序防抖：3帧
- **判定模式**：`达成即过`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `髋部中点` -> 终点 `前膝`
      - 样式：红色
      - 动效：`弹性`

### 错误项 C：躯干重心侧倾
- **ID**：`lean`
- **优先级**：`P3`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`躯干侧倾偏移率`
  - 阈值：`> 0.10`
  - 容忍度：`严格 > 0.08 / 标准 > 0.10 / 宽松 > 0.15`
  - 引导词:
    - `> 0.10 : "身体回正，不要侧倾"`
    - `<= 0.10 : "躯干正直，平衡感很好"`
- **判定策略配置**：
  - 检测时机：全程
  - 单双侧逻辑：整体
  - 时序防抖：5帧
- **判定模式**：`一票否决`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `肩部中点` -> 指向 `目标线`
      - 样式：红色
      - 动效：`弹性`
    - 元素名称：
      - 类型：`目标线`
      - 定义：`髋部中点` -> `垂直向上`
      - 样式：绿色、虚线
