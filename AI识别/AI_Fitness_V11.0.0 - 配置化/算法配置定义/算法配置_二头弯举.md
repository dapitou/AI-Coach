# 算法配置：二头弯举

## 1. 基本信息
- **动作ID**：`bicep_curl`
- **动作名称**：二头弯举
- **版本**：V1.0
- **描述**：检测二头弯举过程中的弯举幅度、躯干借力及肘部漂移情况。

## 2. 全局逻辑控制
- **显示模式**：互斥
- **优先级排序**：升序
- **优先级压制**：开启
- **优先级挤占**：关闭
- **纠错交互触发频率**：2

## 3. 虚拟计算点池配置
- **髋部中点**：
   - *算法*：`中点`
   - *输入*：左髋, 右髋
- **肩部中点**：
   - *算法*：`中点`
   - *输入*：左肩, 右肩

## 4. 指标池配置
- **肩腕距离**：
  - *算法*：`距离`
  - *输入*：肩关节, 腕关节
  - *参数*：模式=欧氏
- **弯举压缩率**：
  - *算法*：`压缩率`
  - *输入*：当前=肩腕距离, 基准=最大臂长基准
- **躯干垂直角**：
  - *算法*：`角度`
  - *输入*：[髋部中点, 肩部中点]
  - *参数*：模式=垂直
- **躯干投影长**：
  - *算法*：`距离`
  - *输入*：肩部中点, 髋部中点
  - *参数*：模式=垂直
- **躯干投影长压缩率**：
  - *算法*：`压缩率`
  - *输入*：当前=躯干投影长, 基准=躯干长度基准
- **躯干晃动率**：
  - *算法*：`波动`
  - *输入*：数值=躯干垂直角, 窗口=5帧
- **肘部前移量**：
  - *算法*：`点线距离`
  - *输入*：点=肘关节, 线=[肩关节, 髋关节]

## 5. 动态基准池配置
- **最大臂长基准**：
  - *名称*：最大臂长
  - *来源指标*：肩腕距离
  - *采集逻辑*：
    - 阻尼：0.05
    - 衰减：0.9995
- **躯干长度基准**：
  - *名称*：躯干长度
  - *来源指标*：躯干投影长
  - *采集逻辑*：
    - 阻尼：0.05
    - 衰减：0.9995

## 6. 状态机配置
- **准备**：
  - 说明：站立姿态，双臂自然下垂
  - 示例：`进入: 复位状态停留时长 > 0.5s`
- **向心 (弯举)**：
  - 说明：弯举阶段
  - 示例：`弯举压缩率 < 0.85`
- **离心 (下放)**：
  - 说明：下放阶段
  - 示例：`弯举压缩率 > 0.90`
- **进入纠错区间**：
  - 说明：进入弯举顶峰
  - 示例：`弯举压缩率 < 0.40`
- **离开纠错区间**：
  - 说明：离开弯举顶峰
  - 示例：`弯举压缩率 > 0.45`
- **复位**：
  - 说明：动作结束
  - 示例：`进入: 弯举压缩率 > 0.95`
- **超时熔断**：
  - 示例：`时长 > 6.0s AND 弯举压缩率 < 0.50`

## 7. 错误项配置架构

### 错误项 A：抬腕幅度不足
- **ID**：`range`
- **优先级**：`P1`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`弯举压缩率`
  - 阈值：`< 0.35`
  - 容忍度：`严格 0.30 / 标准 0.35 / 宽松 0.40`
- **判定策略配置**：
  - 检测时机：纠错区间
  - 单双侧逻辑：分侧
    - 触发逻辑：任一侧不达标
    - 解除逻辑：双侧均达标
  - 时序防抖：3帧
- **判定模式**：`达成即过`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `左腕` -> 终点 `左肩`
      - 方向：`指向目标`
      - 样式：红色
      - 动效：`弹性`

### 错误项 B：躯干借力
- **ID**：`swing`
- **优先级**：`P2`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`[躯干晃动率, 躯干投影长压缩率]`
  - 阈值：`[> 0.10, > 0.10]`
  - 组合逻辑：`OR`
  - 容忍度：`严格 [> 0.08, > 0.08] / 标准 [> 0.10, > 0.10] / 宽松 [> 0.15, > 0.15]`
- **判定策略配置**：
  - 检测时机：全程
  - 单双侧逻辑：整体
  - 时序防抖：5帧
- **判定模式**：`一票否决`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`圆点`
      - 位置：`髋部中点`
      - 样式：红色、实心
      - 动效：`呼吸`

### 错误项 C：肘部漂移
- **ID**：`elbow_drift`
- **优先级**：`P2`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`肘部前移量`
  - 阈值：`> 0.15`
  - 容忍度：`严格 > 0.10 / 标准 > 0.15 / 宽松 > 0.20`
- **判定策略配置**：
  - 检测时机：全程
  - 单双侧逻辑：分侧
    - 触发逻辑：任一侧不达标
    - 解除逻辑：双侧均达标
  - 时序防抖：5帧
- **判定模式**：`一票否决`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `左肘`，方向 `垂直向下`
      - 样式：红色
      - 内容：大臂夹紧身体，不要前后移动！
