# 算法配置：俯身划船

## 1. 基本信息
- **动作ID**：`bent_over_row`
- **动作名称**：俯身划船
- **版本**：V1.0
- **描述**：检测俯身划船过程中的背部平直度、拉起幅度、躯干稳定性及弓背情况。

## 2. 全局逻辑控制
- **显示模式**：互斥
- **优先级排序**：升序
- **优先级压制**：关闭
- **优先级挤占**：关闭
- **纠错交互触发频率**：2

## 3. 虚拟计算点池配置
- **髋部中点**：
   - *算法*：`中点`
   - *输入*：左髋, 右髋
- **肩部中点**：
   - *算法*：`中点`
   - *输入*：左肩, 右肩
- **耳中点**：
   - *算法*：`中点`
   - *输入*：左耳, 右耳

## 4. 指标池配置
- **肩腕距离**：
  - *算法*：`距离`
  - *输入*：肩关节, 腕关节
  - *参数*：模式=欧氏
- **臂长压缩率**：
  - *算法*：`压缩率`
  - *输入*：当前=肩腕距离, 基准=最大臂长基准
- **躯干垂直角**：
  - *算法*：`角度`
  - *输入*：[髋部中点, 肩部中点]
  - *参数*：模式=垂直
- **躯干投影长**：
  - *算法*：`距离`
  - *输入*：肩部中点, 髋部中点
  - *参数*：模式=垂直
- **躯干投影长压缩率**：
  - *算法*：`压缩率`
  - *输入*：当前=躯干投影长, 基准=躯干长度基准
- **躯干晃动率**：
  - *算法*：`波动`
  - *输入*：数值=躯干垂直角, 窗口=5帧
- **脊柱弓曲率**：
  - *算法*：`点线距离`
  - *输入*：点=肩部中点, 线=[耳中点, 髋部中点]
- **肘部屈曲角**：
  - *算法*：`角度`
  - *输入*：[肩关节, 肘关节, 腕关节]
  - *参数*：模式=内角

## 5. 动态基准池配置
- **最大臂长基准**：
  - *名称*：最大臂长
  - *来源指标*：肩腕距离
  - *采集逻辑*：
    - 阻尼：0.05
    - 衰减：0.9995
- **躯干长度基准**：
  - *名称*：躯干长度
  - *来源指标*：躯干投影长
  - *采集逻辑*：
    - 阻尼：0.05
    - 衰减：0.9995

## 6. 状态机配置
- **准备**：
  - 说明：俯身姿态，双臂自然下垂
  - 示例：`进入: 复位状态停留时长 > 0.5s`
- **向心 (拉起)**：
  - 说明：拉起阶段
  - 示例：`肘部屈曲角 < 120°`
- **离心 (下放)**：
  - 说明：下放阶段
  - 示例：`肘部屈曲角 > 140°`
- **进入纠错区间**：
  - 说明：进入划船顶峰
  - 示例：`肘部屈曲角 < 90°`
- **离开纠错区间**：
  - 说明：离开划船顶峰
  - 示例：`肘部屈曲角 > 100°`
- **复位**：
  - 说明：动作结束
  - 示例：`进入: 躯干垂直角 < 20°`
- **超时熔断**：
  - 示例：`时长 > 8.0s`

## 7. 错误项配置架构

### 错误项 A：抬腕幅度不足
- **ID**：`range`
- **优先级**：`P1`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`臂长压缩率`
  - 阈值：`< 0.60`
  - 容忍度：`严格 0.55 / 标准 0.60 / 宽松 0.65`
  - 引导词:
    - `< 0.60 : "再拉高一点，挤压背部"`
    - `>= 0.60 : "幅度完美，控制背部力量下放"`
- **判定策略配置**：
  - 检测时机：纠错区间
  - 单双侧逻辑：分侧
    - 触发逻辑：任一侧不达标
    - 解除逻辑：双侧均达标
  - 时序防抖：3帧
- **判定模式**：`达成即过`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `左腕` -> 终点 `左髋`
      - 方向：`指向目标`
      - 样式：红色
      - 动效：`弹性`

### 错误项 B：躯干借力
- **ID**：`swing`
- **优先级**：`P2`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`[躯干晃动率, 躯干投影长压缩率]`
  - 阈值：`[> 0.15, > 0.15]`
  - 组合逻辑：`OR`
  - 容忍度：`严格 [> 0.10, > 0.10] / 标准 [> 0.15, > 0.15] / 宽松 [> 0.20, > 0.20]`
  - 引导词:
    - `> 0.15 : "核心收紧，身体不要晃"`
    - `<= 0.15 : "躯干很稳，继续保持"`
- **判定策略配置**：
  - 检测时机：全程
  - 单双侧逻辑：整体
  - 时序防抖：5帧
- **判定模式**：`一票否决`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`圆点`
      - 位置：`髋部中点`
      - 样式：红色、实心
      - 动效：`呼吸`

### 错误项 C：弓背
- **ID**：`rounded_back`
- **优先级**：`P1`
- **算法分类**：标准指标型
- **核心算法配置**：
  - 目标指标：`脊柱弓曲率`
  - 阈值：`> 0.05`
  - 容忍度：`严格 > 0.03 / 标准 > 0.05 / 宽松 > 0.08`
  - 引导词:
    - `> 0.05 : "挺胸抬头，背部挺直"`
    - `<= 0.05 : "背部挺得很直，很好"`
- **判定策略配置**：
  - 检测时机：全程
  - 单双侧逻辑：整体
  - 时序防抖：5帧
- **判定模式**：`一票否决`
- **交互反馈配置**：
  - 状态：错误
    - 元素名称：
      - 类型：`箭头`
      - 定义：起点 `肩部中点`，方向 `垂直向上`
      - 样式：红色
      - 内容：背部挺直，不要弓腰！
