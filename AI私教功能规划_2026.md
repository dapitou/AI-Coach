# AI私教功能规划_2026

## AI数据
- 数据采集（感知层）
  - 原始数据采集
    - 用户数据（用户输入）
    - 电机数据（重量、行程等）
    - 摄像头数据（骨骼点、边缘轮廓）
    - 心率带数据（实时心率）
    - 体脂秤数据（身体成分）
  - 三方数据接入
    - 手表端APP（Apple手表，多种手表接入考虑）
    - 实时数据同步（实时心率、动态卡路里等）
    - 异步数据导入（睡眠时长、静息心率等）
  - 行为数据埋点
    - 浏览行为（页面停留、功能点击等）
    - 运动行为（动作跳过、替换等）

- 数据资产（数据层）
  - 用户档案
    - 基础信息（性别、邮箱等）
    - 偏好数据（运动频率、运动类型等）
    - 目标数据（运动目标、目标体重等）
    - 风险数据（伤病历史、禁忌症等）（新增）
  - 运动数据库
    - 运动历史（课程记录、计划进展等）
    - 运动日志（负荷数据、轨迹数据等）
  - 健康数据库
    - 体测历史（各项数据、时间）
    - 身体成分（体重、体脂等）
    - 身体维度（视觉估算、手动记录）（新增）
    - 体姿体态（体态等）
    - 肌肉力量（最大力量、爆发力等）
    - 心肺能力（静息、恢复心率等）
    - 身体柔韧（关节活动度）
    - 动作模式（动作平衡等）
  - 数据治理
    - 数据合规（隐私加密、数据脱敏、本地化存储）
    - 数据清洗（异常值过滤、缺失值填充）
    - 数据精度（传感器校准、算法置信度、多源数据对齐）

- 数据计算（决策层）
  - 用户画像（辅助推荐）
    - 用户标签（人群属性、运动水平等）
    - 运动偏好模型（喜欢/厌恶的动作、强度偏好等）
  - 状态评估模型
    - 肌肉状态（帮助判断今天该训练什么部位）
      - 疲劳维度（肌群疲劳度、恢复预测等）
      - 表现维度（神经募集、代谢压力等）
      - 风险维度（损伤风险）
    - 恢复状态（睡眠质量、压力指数、生理周期等）
    - 力量分数（最大力量、爆发力、肌耐力等）
  - 派生数据计算（基于原始数据计算）
    - 消耗估算（卡路里等）
    - 运动数据（动作次数等）
  - 数据解读引擎
    - 运动表现分析（动作标准度、节奏控制等）
    - 体测数据解读（身体成分分析、改善建议等）

- 数据应用（交互层）
  - 实时数据
    - 实时图表（实时心率/实时功率/速度/实时卡路里/动作计数/倒计时）
  - 数据中心
    - 数据看板（运动概览/目标进度/核心指标）
    - 运动进展（指标趋势/力量增长）
    - 运动偏好（课程类型分布）
    - 肌肉状态（哪些肌肉可练）（新增）
    - 3D数字孪生（直观展示身体状态）（新增）
  - 课程结算（随心练/动作库/跟练课）
  - 计划结算（阶段结算/完成结算）
  - 运动周报
  - 体测报告

## AI推荐
- 推荐算法
  - 内容推荐算法
    - 课程推荐
      - 智能生成（AI推荐）
      - 动态调整（课程中替换动作）
    - 计划推荐
      - 智能生成（AI推荐）
      - 场景推荐（体测结束后）
      - 动态调整（单课/阶段结束触发计划调整）
    - 动作推荐
      - 智能替换（进/退阶动作替换等）
    - 信息流推荐
      - 列表排序（计划、课程、动作）
  - 重量推荐算法
    - 首次重量推荐（体测后推荐）
      - 体测部位1RM
      - 近期训练（长期未训练降力）
      - 练前RPE（状态判断）（新增）
    - 动态重量调整
      - 运动表现（速度衰减、力竭判断、中途失败）
      - 组间RPE（状态判断）（新增）
  - 饮食推荐算法
    - 饮食目标（热量、营养等）
    - 饮食建议（食谱生成、补剂推荐等）
  - 休息推荐算法
    - 睡眠建议（睡眠时间、时长）
    - 肌肉恢复（休息时长）
  
- 推荐交互
  - 推荐生成（自然语言对话式生成）
    - 见AI Agent
  - 生成详情（课程、计划、动作）
    - 推荐理由（为什么推荐）

## AEKE Agent
- 大模型
  - 大模型接入（LLM API集成、Prompt工程等）
  - 意图识别（用户自然语言意图）
  - RAG知识库检索（向量数据库、语义搜索等）

- Agent交互
  - 状态特效（语音唤醒、思考、回答动画）
  - 多模态交互
    - 选项对话（卡片选择）
    - 语音对话（新增）（ASR、TTS、流式交互）
    - 文字对话（新增）（IM交互、上下文记录）
    - 图文对话（新增）（图片/视频识别、富文本回复）
  - 设备控制（开关机、亮度、音量等，硬件指令）
  - 功能调用（浏览课程、开始课程、结束课程等）
  - 运动陪伴（加油鼓励、休息闲聊、情绪价值）
  - 售后咨询（故障诊断/客服工单对接）
